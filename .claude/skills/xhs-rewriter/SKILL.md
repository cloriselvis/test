# 小红书爆款改写器

## 核心任务

将X推文改写为符合小红书平台调性的高质量内容，最大化流量和互动。

## 小红书内容公式

```
标题（吸睛） + 开头（钩子） + 正文（价值） + 结尾（互动） = 爆款
```

---

## 第一步：标题创作（最重要！）

小红书80%的流量来自标题，必须在1秒内抓住用户。

### 标题公式（必选其一）

1. **数字冲击型**
   - "3个月赚47万！我的Notion模板副业全流程"
   - "实测！这5个AI工具帮我月入2万+"
   
2. **痛点共鸣型**
   - "还在死工资？这个副业让我辞职了"
   - "别再被割韭菜！自媒体变现的5个真相"

3. **利益承诺型**
   - "0成本起步的3个副业，我选了第2个"
   - "新手也能做！不露脸的赚钱方法"

4. **反常识型**
   - "副业比主业赚得多？我的真实经历"
   - "不建议做跨境电商的3个理由（血泪教训）"

5. **清单列表型**
   - "副业避坑指南｜这10个坑我都踩过"
   - "2024最火的7个线上副业，亲测有效"

### 标题撰写规则

✅ **必须做：**
- 加入数字：收入数字、时间、步骤数、清单数
- 使用emoji：💰🔥⚠️✅❌ 等（但不要超过3个）
- 控制字数：15-25字最佳，不超过30字
- 制造冲击：真实数据、反差对比、颠覆认知

❌ **禁止：**
- 全大写、多个感叹号（！！！会被限流）
- 夸大承诺："日入1000""躺赚""暴富"（平台打击）
- 标题党无实质内容（会被举报）
- 敏感词：金融、投资、理财、股票（容易违规）

### 标题模板库（直接套用）

**副业赚钱类：**
- "[收入数字] | [时间周期] + [方法名称] + 真实分享"
  - 示例："月入3万｜我的AI写作副业全拆解"
  
- "[身份标签] 靠 [副业类型] 实现了 [结果]"
  - 示例："95后宝妈靠小红书运营赚到第一个10万"

- "[数字]个 [副业/工具/方法] + [效果/避坑]"
  - 示例："5个不需要露脸的副业，最后1个我做了2年"

**干货教程类：**
- "[主题] 保姆级教程 | [受众] 必看"
  - 示例："ChatGPT变现保姆级教程｜新手3天上手"

- "实测 [工具/方法] | [时间] 后的真实效果"
  - 示例："实测小红书引流｜30天涨粉5000的方法"

**避坑类：**
- "别 [错误行为]！[正确方法] 才是关键"
  - 示例："别盲目做跨境！先搞清楚这3件事"

- "[话题] 的 [数字] 个真相，第 [数字] 个扎心了"
  - 示例："自媒体变现的5个真相，第3个让我亏了2万"

## 第二步：开头设计（前3行决定完读率）

小红书用户耐心极短，前3行必须留住人。

### 开头公式（3选1）

**1. 数据震撼型（最推荐）**
```
💰 3个月收入47万
💼 零成本起步
⏰ 每天只花2小时

这是我做Notion模板副业的真实数据
（不是标题党，收入截图在最后👇）
```

**2. 痛点共鸣型**
```
还在每天早上挤地铁上班吗？
我也是这样熬了3年...
直到我发现这个方法，现在在家就能赚钱

今天全部分享给你们 👇
```

**3. 故事代入型**
```
去年这个时候，我还是个月薪5k的打工人
现在副业收入是主业的3倍

不是运气好，是找对了方法
我的完整经历都在这里了 ⬇️
```

### 开头撰写规则

✅ **必须做：**
- 第一句话就给强刺激（数字/痛点/结果）
- 制造悬念："方法在后面""截图在文末"
- 使用换行：每1-2句就换行，增加可读性
- 添加箭头emoji：👇⬇️ 引导继续阅读

❌ **禁止：**
- 长篇大论的背景介绍（直接进主题）
- 废话铺垫："大家好，我是XXX"（没人关心）
- 过度谦虚："我也不是专家，随便分享下"（降低信任）

## 第三步：正文结构（价值交付）

小红书用户要的是"拿来就能用"的干货。

### 内容组织原则

**分点呈现，不要大段文字**
- 用 ✅ ❌ 💡 📌 等符号做视觉区隔
- 每个要点2-3行，然后换行
- 核心信息加粗（但不要过度）

**信息密度适中**
- X推文往往很精简，需要补充案例和细节
- 但不要写成长文，小红书最佳正文长度800-1500字
- 能用清单就不用段落

### 正文模板

**方法论类（拆解步骤）：**
```
我的完整流程分享 👇

第一步：[步骤名称]
• [具体操作1]
• [具体操作2]
💡 Tips：[关键技巧/避坑提醒]

第二步：[步骤名称]
• [具体操作1]
• [具体操作2]
⚠️ 注意：[常见错误]

第三步：[步骤名称]
...

【我的真实效果】
[配数据截图/案例说明]
```

**工具/资源推荐类：**
```
我筛选了50+工具，最推荐这5个 👇

1️⃣ [工具名称] - [一句话介绍]
✅ 优点：[2-3个]
❌ 缺点：[诚实说明]
💰 价格：免费 / XX元
🔗 [如何获取]

2️⃣ [工具名称]
...

【我的使用经验】
[个人心得/避坑建议]
```

**经验分享类：**
```
我踩过的坑 vs 正确做法 👇

❌ 错误示范：[我以前怎么做的]
后果：[亏了多少/浪费多少时间]

✅ 正确方法：[现在怎么做]
效果：[具体改善数据]

[重复2-3组]

💡 核心心得：
[总结提炼的方法论]
```

### 正文撰写规则

✅ **必须做：**
- 多用短句：一句话不超过20字
- 高频换行：2-3行就换行，不要大段文字
- 具体化：不说"很多"，说"47个"；不说"很快"，说"3天"
- 真实性：配截图、数据、案例（小红书用户很精明）
- 降低门槛：X推文如果用了专业术语，一定要解释

❌ **禁止：**
- 空洞的鸡汤："只要努力就能成功"
- 过度谦虚："个人看法，不一定对"（直接说干货）
- 纯理论："根据XX理论，可以得出..."（小红书不是论文）
- 攻击性言论：即使X原文有，也要改得温和

## 第四步：结尾设计（互动转化）

结尾决定你的涨粉和收藏率。

### 结尾公式（必选其一）

**1. 福利钩子型（涨粉首选）**
```
🎁 福利时间！

完整版资料包（包含XX工具清单+XX模板）
👉 评论区扣"1"，我私发给你
或者直接关注我，主页有领取方式

记得点赞收藏，怕你们找不到了～
```

**2. 互动引导型（提升评论）**
```
💬 互动一下！

你们现在在做什么副业？
收入怎么样？评论区聊聊呗

我会逐个回复的💕
点赞的都能月入过万（玄学哈哈）
```

**3. 系列预告型（提升关注）**
```
这篇讲了XX方法
下期我会详细拆解XX案例

关注我 @[用户名]
持续分享副业变现干货

不迷路不走丢 ✨
```

**4. 真诚分享型（建立信任）**
```
以上就是我3个月的真实经历
有问题欢迎评论区问我

希望能帮到同样在摸索副业的你们
一起加油吧 💪

觉得有用就点个赞收藏
你的支持是我分享的动力～
```

### 结尾撰写规则

✅ **必须做：**
- 给明确行动指令："评论扣1""关注我""点赞收藏"
- 降低行动成本：不要让用户"转发朋友圈""填写问卷"（太复杂）
- 真诚语气：小红书用户喜欢真诚，不喜欢营销感
- emoji点缀：💕✨🎁等暖色系emoji

❌ **禁止：**
- 硬广："加我微信""买我课程"（会被限流）
- 乞讨式："求求你们点赞吧"（掉价）
- 威胁式："不点赞就取关"（反感）
- 空结尾：没有任何互动引导（浪费流量）

## 第五步：全文优化检查清单

在输出最终版本前，必须检查：

### ✅ 平台规范检查
- [ ] 无违禁词：暴富、躺赚、日入、内幕、黑科技
- [ ] 无敏感词：金融、股票、币圈、博彩、贷款
- [ ] 无硬广：微信号、二维码、外链
- [ ] 无过度承诺：100%成功、保证赚钱、无风险

### ✅ 内容质量检查
- [ ] 标题有吸引力（数字+emoji+利益点）
- [ ] 开头3行有冲击力（数据/痛点/悬念）
- [ ] 正文有实质价值（具体方法/工具/数据）
- [ ] 结尾有互动引导（评论/关注/点赞）

### ✅ 格式美观检查
- [ ] 每2-3行就换行（不要大段文字）
- [ ] 有emoji点缀（但不超过全文的5%）
- [ ] 有符号区隔（✅❌💡📌等）
- [ ] 总字数800-1500字（太短没价值，太长没人看）

### ✅ 真实性检查
- [ ] 收入数字合理（不要夸大10倍）
- [ ] 方法可复制（不是靠运气或资源）
- [ ] 有踩坑提醒（显得真诚可信）
- [ ] 语气接地气（不要端着）

## 特殊处理规则

### 金额换算
- X推文如果是美元，换算成人民币（汇率7.2左右）
- 标注清楚："47K美金≈34万人民币"或直接写人民币

### 文化本地化
- 国外平台换成国内平台：Reddit → 豆瓣/知乎，Twitter → 微博/小红书
- 工具替换：Gumroad → 微店/有赞（如果可替代）
- 保留原平台如果不可替代，但加注释

### 时效性处理
- X推文如果有"2023"等时间，改为"去年"或"最近"
- 数据更新："最新数据显示..."（即使用的是旧数据）

### 争议内容软化
- X上激进观点 → 温和表达："我的个人看法是..."
- 批评某公司/产品 → 改为"某平台""某工具"
- 删除攻击性语言，保持价值观中立
